# ✊ RPS AI - Rock Paper Scissors

An AI-powered Rock Paper Scissors game where the computer learns and predicts
user moves using machine learning and pattern analysis. The system analyzes
the player's previous choices to identify behavioral patterns and adapts its
strategy in real time to improve win probability against the user.


## ✨ Features

- ✊ AI learns and predicts your next move
- 🧠 Pattern analysis from previous choices
- 🎮 Real-time adaptive strategy
- 📊 Win / Loss / Draw score tracker
- ⚡ Instant AI response every round


## 🧠 Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
- Streamlit


## 📊 Models Used

- Pattern Recognition Algorithm
- Markov Chain for move prediction
- Frequency Analysis


## ▶️ Run Locally

git clone https://github.com/dheeraj815/RPS-AI.git
cd RPS-AI
pip install -r requirements.txt
streamlit run app.py
